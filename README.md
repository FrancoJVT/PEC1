# Playas de Okinawa 

Proyecto web desarrollado como parte de la **PEC1 de HTML y CSS II** (UOC).  
Se muestra la información sobre las playas de Okinawa, Japón, con secciones de actividades, galería de imágenes y guía de visita.

---

## 🔹 Características del proyecto

- **HTML5 y CSS3** modernos, con estructura semántica
- **Sass/SCSS** para estilos organizados
- **JavaScript** básico para interacción
- **Parcel** como bundler, con compilación, minificación y optimización
- **Galería de imágenes** y sección de actividades
- **Responsive design**: adaptable a móviles, tablet y escritorio
- **Favicon** y logotipo integrados

---

## Instalación

1. Clonar el repositorio:

```bash
git clone git@github.com:FrancoJVT/PEC1.git
cd PEC1
```

2. Instalar dependencias:
```bash
npm install
```
Levantar servidor de desarrollo:
```bash
npm run dev
```
Generar versión de producción (archivos compilados en dist/):
```bash
npm run build
```
